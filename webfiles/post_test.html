Project 2 Test Page for POST Method

Use this page to test optional support for the POST method. Fill out the following form and press the "Submit Query" button button below. Your server will receive a POST request and should save the information in the file post_test_file.txt in the base directory for your web server.

Note that /post_test_file.txt is the Request-URI that will be provided with the POST request. Your server must not hard-code this file name. Instead, the server should save the posted data to the file specified in the Request-URI that comes with the POST request.
